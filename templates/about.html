Technology Used

This project was created using the following technology

  • Python
  • Flask
  • SQL-Alchemy
  • Bootstrap
  • PopperJs
  • jQuery

Libraries such as url lib were used for parsing Youtube urls. This allowed for 3 different version of a link for a YouTube video to be used for the gifsound.

Amazon Web Services RDS for hosting database.

The Giphy API for displaying GIF suggestions on the homepage.
